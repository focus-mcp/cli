feat: namespace `bricks:` for CLI commands, MCP tools renamed to `focus_<namespace>_<action>` pattern. `focus update`/`upgrade` now self-update the CLI (use `bricks:update` for bricks).

BREAKING CHANGES:
- MCP tools `focus_install`, `focus_remove`, `focus_search`, `focus_load`, `focus_unload`, `focus_update`, `focus_upgrade` are renamed to `focus_bricks_*`
- `focus_tools` singleton is split into 6 distinct tools: `focus_tools_hide`, `focus_tools_show`, `focus_tools_pin`, `focus_tools_unpin`, `focus_tools_list`, `focus_tools_clear`
- New MCP tool `focus_self_update` returns the CLI update command to run
- `focus update <brick>` no longer works — use `focus bricks:update <name>`
- `focus update` / `focus upgrade` now self-update the CLI (not bricks)
- `focus update --all` self-updates CLI + all bricks
